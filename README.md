# 🎮 Simon Says Game

## 🧾 Overview
This is a simple **Simon Says memory game** built using HTML, CSS, and JavaScript.  
The game tests the player's memory by generating a random sequence of colors that the player must repeat in the correct order.

---

## ✨ Features
- Random sequence generation  
- Increasing difficulty level with each round  
- Interactive button-based UI  
- Visual feedback (button flash effects)  
- Game over detection on wrong input  
- Restart option to play again  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

## 🎯 How to Play
- Press any key / Start button to begin the game  
- Watch the sequence of colors shown by the game  
- Repeat the same sequence by clicking the buttons  
- Each level adds a new color to the sequence  
- If you click the wrong color → Game Over  
- Try to reach the highest level possible
